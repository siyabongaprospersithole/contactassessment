## Contacts App

Installation:

- Run composer update
- Run npm i
- Run npm run dev
- OPen .env file to config datatabase
- Run Migrations
- php artisan route:list -- view all routes
- Config Email
- Serve the Application

Seeders:
- php artisan db:seed --class=GendersSeeder
